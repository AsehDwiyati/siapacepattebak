# SiapaCepatTebak - Game Real-Time

## Cara Deploy ke Render (gratis):
1. Upload semua file ini ke GitHub repository
2. Masuk ke https://render.com
3. Login dan klik "New Web Service"
4. Pilih repo GitHub yang berisi folder ini
5. Isi Build Command: `npm install`
6. Isi Start Command: `npm start`
7. Tunggu Render menyelesaikan build
8. Kamu akan dapat URL seperti:
   https://siapacepattebak.onrender.com

9. (Opsional) Hubungkan domain dari Hostinger ke URL ini via DNS

Enjoy!
